% = % of squat 1RM  ·  RPE = rate of perceived exertion (log how hard it felt, 6–10)  ·  @X0X1 = tempo (ecc / pause / con / pause)
Deload weeks (wk 4 of each block) run ~60% volume — drop a set, keep the bar moving, prioritise sleep.
SHLD = shoulder-conscious movement; stop anything that pinches the right shoulder, regress range as needed. No overhead pressing until your rehab clears it.
Plyos always done FRESH and never to failure — quality of contact > quantity. If a jump slows or sloppy, the set is over.
Data lives in this file only. Export regularly to keep your log. — Built for a Muay Thai / calisthenics base returning through shoulder rehab.
